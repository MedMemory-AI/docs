# 🛠️ Tech Stack Overview

| System Layer | Technology Pick | Purpose |
| :--- | :--- | :--- |
| **Backend API Server** | FastAPI (Python) | Async routing, core business logic |
| **Mobile Interface** | React Native | Cross-platform frontend application |
| **Orchestration Core** | LangChain & LangGraph | Agent cycles, RAG toolchain mechanics |
| **Embedding Generation** | `mxbai-embed-large` | 1024-dim dense clinical semantic vectors |
| **Local LLM Model** | `llama3.2` | Localized Pydantic JSON entity extraction |
| **Relational Storage** | PostgreSQL | Timeline indexing, structural attributes |
| **Vector Indexing** | Qdrant DB | Dense spatial vector search |
